# Smart Contract

## Cosa è uno Smart Contract?

Uno smart contract è un programma che viene eseguito sul blockchain. Il programma è scritto in un linguaggio di programmazione specifico per la blockchain, come Solidity. Ci sono vari tipi di standard come ERC721, ERC20, i quali sono i più famosi per la creazione di token.

## Codice template

```solidity filename="Counter.sol" copy

pragma solidity ^0.5.0;

contract Counter {
    uint256 public count;

    function increment() public {
        count += 1;
    }
}
```
